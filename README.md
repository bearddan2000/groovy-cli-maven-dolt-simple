# groovy-cli-maven-dolt-simple

## Description
A groovy maven build, that connects to dolt database.

## Tech stack
- groovy
- maven
  - 6.8.2

## Docker stack
- dolthub/dolt-sql-serverdb:latest
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
